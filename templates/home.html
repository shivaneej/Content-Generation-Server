About

Every article or formal letter consists of key points. While writing an article, the writer formulates the key points in his/her head and forms the article around those key points. Genessay, a text completion software, will assist the user in this process by taking the title and the key points of the article as its input and formulate the article based on the key points using Natural Language Processing and Information Retrieval from external sources. This will also be implemented for generating application letters using machine learning algorithms. An additional feature for optimizing the text will improve the entered content by using better vocabulary and correcting grammatical errors. Genessay should guarantee accuracy better than the existing system and should work as a cross-platform software.
